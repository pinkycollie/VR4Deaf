# Agents

This directory contains AI-powered agents for automation, assistance, and intelligent features in the VR4Deaf platform.

## Available Agents

(Modules will be added here as they are developed)

### Planned Agents

- **Virtual Support Agent** (`@vr4deaf/magician-agent-support`)
  - 24/7 automated assistance
  - Visual-first responses
  - ASL interpretation
  - Context-aware help

- **Career Matching Agent** (`@vr4deaf/magician-agent-career`)
  - Skills assessment
  - Job matching algorithms
  - Career path recommendations
  - Accessibility-aware matching

- **Learning Assistant** (`@vr4deaf/magician-agent-learning`)
  - Adaptive learning guidance
  - Visual-first tutoring
  - Progress tracking
  - Personalized recommendations
