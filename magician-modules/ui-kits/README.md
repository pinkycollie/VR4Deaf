# UI Kits

This directory contains reusable UI component libraries for building Deaf-first applications with visual-first design, ASL integration, and high-contrast accessibility.

## Available Kits

(Modules will be added here as they are developed)

### Planned UI Kits

- **ASL Navigation** (`@vr4deaf/magician-ui-asl-navigation`)
  - Visual-first navigation components
  - Icon-based menus
  - Gesture-friendly interactions
  - ASL-interpreted labels

- **Accessibility Components** (`@vr4deaf/magician-ui-accessibility`)
  - Visual alerts and notifications
  - High-contrast UI elements
  - Captioned media players
  - Keyboard navigation support

- **Communication Kit** (`@vr4deaf/magician-ui-communication`)
  - Real-time captioning components
  - Visual feedback indicators
  - ASL video integration
  - Visual notification systems
